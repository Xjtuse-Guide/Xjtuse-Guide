## 06

### 重写对新方法的限制如下：
- 参数列表必须与继承的方法完全一致。
- 返回类型必须与继承的方法相同。
- 访问修饰符不能比继承方法的修饰符更严格。

### 方法查找
- 编译器从对象的类定义开始查找方法调用的实现。
- 如果在对象的类中未找到该方法，查找会继续在其超类中进行，直到找到为止。
- 当方法被找到时，它将在接收到消息的对象上被调用。
- 如果在类层次结构中的任何类中都没有实现该方法，编译时会报错。

### 继承与静态方法
- 一个类可以像调用自身定义的静态方法一样调用其超类中定义的所有静态方法。
- 静态方法可以被子类中的静态方法隐藏。

### 关于 this 和 super
- `this` 和 `super` 可以在实例方法和构造函数中使用，但不能在类（静态）方法中使用。
- `this` 和 `super` 会影响方法查找。
- `this`：方法查找从当前类开始。
- `super`：方法查找从直接的超类开始。
- `super` 关键字允许你在当前类中重写方法时使用来自超类的代码。
- 当子类重写了超类的方法时，调用该方法会执行重写的超类方法的代码。
- 使用 `super` 允许你从直接的超类开始查找方法。
