## 02

### 标识符是：
- 用于表示变量、方法、类或标签的文本字符串
- 区分大小写
- 可以包含数字、字母、`$` 或 `_` 字符

### 标识符不能：
- 以数字开头
- 与保留字相同

### 命名约定：
- **包名**：全部小写
    - 例如：`theexample`
- **类名**：首字母大写，复合词中每个单词首字母大写
    - 例如：`TheExample`
- **方法名 / 字段名**：首字母小写，复合词中后续单词首字母大写
    - 例如：`theExample`
- **常量名**：全部大写，单词之间用下划线分隔
    - 例如：`THE_EXAMPLE`

### Java 基本数据类型（primitives）
- 每个变量都必须有一个数据类型
- 基本数据类型只能包含一个单独的值
- 每种基本数据类型在大小和格式上都适合其特定用途
- Java 的基本数据类型分为四大类（integer，char，float，boolean）

### 字符（char）
- 任何无符号的 Unicode 字符都可以作为 char 基本数据类型
- 字符是放在两个单引号之间的单个 Unicode 字符
    - 例如：`'A'`、`'中'`、`'\n'`
- 默认初始值为 Unicode 编码为 `\u0000` 的字符，即空字符

### 声明和初始化
- 变量必须在使用之前声明
- 单一值变量（即不是数组的变量）必须在首次使用前进行初始化
- 声明和初始化可以结合在一起
- 使用 `=` 进行赋值（包括初始化）

### 作用域
- 变量的作用域是程序中可以引用该变量的区域
- 在方法中声明的变量只能在该方法中访问
- 在循环或代码块中声明的变量只能在该循环或代码块中访问
