- ## 03

    ### 声明和初始化对象
    - 就像基本类型和数组一样，对象在使用之前必须被声明。声明需要对象的类型。类型就是对象的类
    - 使用 `=` 进行赋值（包括初始化）
    - 对象的初始化通常使用 `new` 运算符
    - 一个对象可以被初始化为 `null`
    - 对象数组的声明方式与基本类型数组相同，对象数组默认被初始化为 `null`

    ### 包装类（Wrapper classes）
    - 基本类型没有关联的方法；也就是说，基本数据类型本身不包含任何行为
    - 每一种基本数据类型都有一个对应的类，这种类被称为“包装类”
    - 每一个包装类对象只存储一个基本类型的变量，并提供处理该变量的方法
    - 包装类是 Java 基础 API 的一部分

    - String 类型是一个类，而不是一种基本数据类型

    ### 连接字符串（Concatenating strings）
    - `+` 运算符可以用于连接字符串：
        - 例如：`String a = "This" + " is a " + "String";`
    - 尽管可以这样连接字符串，但还有更高效的方法（将在后面讨论）
    - 当 `println` 方法中使用基本数据类型时，它们会被自动转换为字符串
        - 例如：
        - `System.out.println("answer = " + 1 + 2 + 3);` 输出的是 `"answer = 123"`
        - `System.out.println("answer = " + (1+2+3));` 输出的是 `"answer = 6"`

    ### StringBuffer
    - `StringBuffer` 类提供了一种更高效的构建字符串的机制
    - 字符串连接可能会非常昂贵
    - 大多数编译器会将字符串连接转换为 `StringBuffer` 实现
    - 如果是构建简单的字符串，可以直接连接；如果通过循环构建字符串，则应使用 `StringBuffer`
