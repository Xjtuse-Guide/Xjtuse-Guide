# 2.选择排序

## 2.1 算法原理

选择算法（Selection Sort）的核心思想是：

每次遍历找到“最小元素”，与当轮起始位置交换，确定该位置的最终元素。~~最人性化~~

每次确定一个当前最小元素后，这个元素与后面的所有元素大小关系确定。经过n-1次后即可确定n个元素的大小关系。

## 2.2 代码实现

用`i`控制遍历次数，遍历范围为$[0, n - 1]$。用`minIndex`指向当前最小的元素，向后遍历发现更小的元素的时候更新指针，最后将该元素放到`i`位，即该元素的最终位置。

```java
public static void selectionSort(int arr) {
    int n = arr.length;
    for (int i = 0; i < n - 1; i++) {
        int minIndex = i;
        for (int j = i + 1; j < n; j++) {
            if (arr[j] < arr[minIndex]) {
                minIndex = j;
            }
        }
        swap(arr, i, minIndex);
    }
}
```

## 2.3 复杂度分析

### 2.3.1 时间复杂度

所有情况下时间复杂度都是*O(n²)*。

### 2.3.2 空间复杂度

原地排序，空间复杂度*O(1)*。
